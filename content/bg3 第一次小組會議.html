第三組第一次會議

會議流程:

  1. 直播會議,組員簽到
  2. 討論工作分配 
  3. 問題提問

作業一:

  1. 請描述如何針對該課程進行有效的隨機分組，或者隨機進行點名？
  2.  描述如何為 Windows 10 64 位系統準備一個可移植的 Python 編程系 統，以允許在 Github 上維護 CMSiMDE 網站，Pelican 博客和 Reveal.js 演示文稿？
  3.  您需要從  http://www.coppeliarobotics.com/helpFiles/index.html 了解什 麼來實現四輪機器人？

注意事項:

  1. 作業一必須在 3/26 前完成，並上傳 pdf 檔到個人倉儲的 download 上。
  2.  每周務必上傳進度。
  3.  不會做的可以問會的同學或在 gitter 上發問。
  4.  請依每個人的作業一 pdf 檔、網站內容，進行互評
